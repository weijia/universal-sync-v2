# 更新说明

## 2025-11-03 重大更新

### 完成的工作

1. **集成 zen-fs-webdav**
   - 将 `zen-fs-webdav` 添加为本地依赖
   - 修改 `zen-fs-webdav` 使其完全兼容 `IFileSystem` 接口
   - 添加了 `readdir()` 和 `rename()` 方法以兼容 Node.js fs.promises API
   - 修改 `readFile()` 支持直接传入 encoding 字符串参数

2. **简化浏览器集成**
   - 创建了包含所有依赖的浏览器 bundle (`dist/browser.js`)
   - 移除了复杂的 import map（只需要 PouchDB）
   - 支持两种存储后端：
     - **IndexedDB**：本地浏览器存储
     - **WebDAV**：远程服务器存储

3. **API 改进**
   - `syncBrowser(options)`：简化的一键同步接口
   - `createBrowserFS(options)`：创建文件系统实例，支持手动控制
   - 统一的配置接口

### 现在可以这样使用

```javascript
import PouchDB from 'pouchdb';
import { syncBrowser } from './dist/browser.js';

const db = new PouchDB('mydb');

// 使用 IndexedDB（本地）
await syncBrowser({ db });

// 或使用 WebDAV（远程）
await syncBrowser({ 
    db,
    webdav: {
        url: 'https://webdav.example.com',
        username: 'user',
        password: 'pass'
    }
});
```
